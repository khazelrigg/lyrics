# Project Instructions


## Objective
This is a web based application that allows users to connect to Spotify and view the lyrics for their currently playing song. It is focused on foreign language learners (ex: Japanese/Korean) and the ability to use browser tools like Yomitan or connect to Anki via Ankiconnect.

## Frontend:
- React
- Vite
- TypeScript
- Zustand
- Shadcn UI
- Tailwind v4

## Backend:
- FastAPI
- Python

## Guidelines:
- Prefer existing shadcn components
- Use @ imports for Vite
- Avoid creating duplicate components
- Reuse Zustand stores
- Keep styling consistent with Spotify dark theme
- Prefer functional React components
